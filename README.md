## Module Summaries

### Task 1 – Digital Literacy Awareness Infographic (Module 1)
Created a one-page infographic using **Canva** covering what digital literacy means, useful digital tools for students, and safe internet practices. Exported and added to `task-1-presentation/`.

### Task 2 – Student Digital Portfolio (Module 2)
Set up professional accounts on **GitHub**, **LinkedIn**, and **Kaggle**. Created a GitHub profile README and filled in the Education section on LinkedIn. Screenshots are in `task-2-portfolio/`.

### Task 3 – Coding and Collaboration Platforms (Module 3)
Completed beginner challenges on **HackerRank** (Solve Me First + Python Introduction). Created a 5-question **Digital Literacy Awareness Quiz** using Google Forms, linked to a Google Sheet.

### Task 4 – Professional Email and Etiquette Guide (Module 4)
Wrote two professional emails: a deadline extension request and an internship application. Created a Social Media Do's and Don'ts checklist. Files in `task-4-email-etiquette/`.

### Task 5 – Cybercrime Awareness: UPI Fraud (Module 5)
Researched UPI and Online Payment Fraud. Wrote a 200+ word case study based on realistic fraud scenarios and created a 10-point prevention checklist with UPI-specific safety tips. Files in `task-5-cybercrime/`.
